# Contributing

Prerequisites | Minimum Version
|---|---|
node | ^8.11.x
[yarn](https://yarnpkg.com/en/) | ^1.3.0


1. Fork the repo
2. Clone locally
3. `yarn install`
4. `git checkout -b <your-branch-name>`
5. `cp sample.env .env`
6. `yarn develop`
7. Make your changes
8. Make a PR
9. Bask in the glory of your accomplishments
